邮箱地址转mailto链接

将HTML中的纯文本邮箱地址转换为可点击的mailto链接，增强用户体验并防止爬虫抓取。

原始代码

下面是包含纯文本邮箱地址的HTML代码：

<div class="credits">
    <div class="container">
        <div class="copyright">
            <a href="https://github.com/Lednerb" target="_blank">
                © 2025 Wang Lab. wangjk@mail.sysu.edu.cn
            </a>
            -
            <a href="https://lednerb.github.io/bilberry-hugo-theme/index.xml">rss</a>
        </div>
        <div class="author">
            <a href="https://github.com/Lednerb/bilberry-hugo-theme/tree/v4.13.0"
                target="_blank">Bilberry Hugo Theme v4.13.0</a>
        </div>
    </div>
</div>

问题：邮箱地址 wangjk@mail.sysu.edu.cn 是纯文本形式，用户无法直接点击发送邮件，同时也容易被爬虫抓取。

转换后代码

下面是修改后的代码，邮箱已转换为mailto链接：

<div class="credits">
    <div class="container">
        <div class="copyright">
            <a href="https://github.com/Lednerb" target="_blank">
                © 2025 Wang Lab. </a>
            <a href="mailto:wangjk@mail.sysu.edu.cn" class="mail-link">wangjk@mail.sysu.edu.cn</a>
            -
            <a href="https://lednerb.github.io/bilberry-hugo-theme/index.xml">rss</a>
        </div>
        <div class="author">
            <a href="https://github.com/Lednerb/bilberry-hugo-theme/tree/v4.13.0"
                target="_blank">Bilberry Hugo Theme v4.13.0</a>
        </div>
    </div>
</div>

解决方案：将邮箱地址包裹在 <a href="mailto:wangjk@mail.sysu.edu.cn"> 标签中，使其成为可点击的链接。
